
# 🎼 Quick Composition Guide (Concert Pitch, C Major)

A fast, practical reference for writing simple, coherent music using **melody + harmony + rhythm**, designed for easy import into notation software and later transposition.

---

## 1. Why Work in C Major (Concert Pitch)

* No sharps or flats → faster decisions
* Reduces theory overhead → focus on structure
* Easy to transpose later

**Workflow:**

1. Write everything in C major
2. Make it sound good
3. Transpose for instruments later

---

## 2. Core Scale & Chords

### C Major Scale

C – D – E – F – G – A – B

### Diatonic Chords

| Degree | Chord | Function               |
| ------ | ----- | ---------------------- |
| I      | C     | Home / resolution      |
| ii     | Dm    | Pre-dominant           |
| iii    | Em    | Passing                |
| IV     | F     | Expansion / lift       |
| V      | G     | Tension                |
| vi     | Am    | Relative minor / color |
| vii°   | Bdim  | Leading tone (rare)    |

## 3. Plug-and-Play Chord Progressions

### Default Progression

```
C – G – Am – F
(I – V – vi – IV)
```

### Basic Phrase

```
C – F – G – C
(I – IV – V – I)
```

### Strong Ending

```
Dm – G – C
(ii – V – I)
```

### Slightly Richer

```
Am – Dm – G – C
(vi – ii – V – I)
```

## 4. Melody Writing Rules (High Impact)

### Rule 1: Target Chord Tones on Strong Beats

Strong beats = 1 and 3 (in 4/4)

Chord tones = 1st, 3rd, 5th of chord

Example (C chord):

* Good: C, E, G on strong beats
* Passing tones allowed between

---

### Rule 2: Use Stepwise Motion

* Mostly move by step (C → D → E)
* Use leaps sparingly for emphasis

---

### Rule 3: Shape Phrases

* Start low → rise → resolve
* Think in **4-bar phrases**

---

### Rule 4: Repetition + Variation

Repeat motifs but change:

* Ending note
* Rhythm
* Direction

---

## 5. Supporting Voice (Alto / Left Hand / Harmony Layer)

### Option A: Sustained Harmony (Easiest)

* Whole or half notes
* Play root or 3rd of chord

Example:

* C chord → play C or E
* F chord → play F or A

---

### Option B: Counter-Melody (More Musical)

Rules:

* Move **opposite direction** of melody
* Avoid same rhythm as melody
* Use longer note values

---

### Option C: Broken Chords (Arpeggios)

Instead of holding:

* C → play C–E–G pattern
* Adds motion without clutter

---

## 6. Rhythm Basics (Critical)

### Keep It Simple First

* Use mostly:

  * Quarter notes
  * Eighth notes

---

### Strong vs Weak Beats

| Beat | Strength |
| ---- | -------- |
| 1    | Strong   |
| 2    | Weak     |
| 3    | Medium   |
| 4    | Weak     |

👉 Important notes go on strong beats

---

### Avoid Common Problems

❌ Too many accents
❌ Random syncopation
❌ All notes same length

---

### Use Rhythm for Interest

Instead of changing notes, try:

* Repeat note, change rhythm
* Add rests
* Syncopation (occasionally)

---

## 7. 4-Bar Writing Template (Use This Every Time)

### Step 1: Pick progression

```
Bar 1: C
Bar 2: F
Bar 3: G
Bar 4: C
```

---

### Step 2: Write melody

* Use chord tones on beats 1 & 3
* Fill with passing notes

---

### Step 3: Add harmony

* Whole notes (easy mode)
  or
* Simple counter-line

---

### Step 4: Fix rhythm

* Add variation
* Remove clutter

---

## 8. Fast Checklist

Before you finish:

* ✅ Are strong beats chord tones?
* ✅ Is there a clear phrase (start → peak → end)?
* ✅ Does harmony match melody?
* ✅ Is rhythm consistent but not boring?
* ✅ Does it resolve (not just stop)?

---

## 9. Transposing Later (Sax Example)

After writing in C:

* Bb instrument → up major 2nd
* Eb instrument → up major 6th

Most notation software does this automatically.

---

## 10. Key Principle

> **Clarity beats complexity.**

A simple, well-structured idea will always sound better than a complex but unfocused one.
