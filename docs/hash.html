How to Calculate a SHA-256 Hash

Windows (CMD)

1. Open the Command Prompt.

2. Run this command (replace filename.ext with your downloaded file):

certutil -hashfile filename.ext SHA256

3. The SHA-256 hash will appear on the next line.

Windows (PowerShell)

1. Open PowerShell.

2. Run the following command:

Get-FileHash filename.ext -Algorithm SHA256

macOS

1. Open the Terminal.

2. Run this command:

shasum -a 256 filename.ext
