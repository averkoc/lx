## How to prepare import-file for netacad bulk enrollment   

Copy all students from the downloaded Peppi participants excel-files and create a formula that combines firstname, lastname and email in the netacad required way. See the picture below for the formula. After that copy the G-column values and paste them to notepad++ and save as csv-file.  

![import](../images/importdata.png)  

> Without using the named cell quote you should type three quotes (""") to produce one quote. It would be a messy formula.
